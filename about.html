About Simple Gamez

Simple Gamez is a web gaming portal dedicated to hosting high-quality, instant-play, browser-based games. Our mission is to provide clean, fun, and engaging casual games that load instantly and run smoothly on any device without requiring downloads, installs, or logins.

Our current featured games include:

  • Brain Rot Merge: Drop and evolve iconic memes in a chaotic, bouncy physics box!
  • Raising Comet: Guide your comet through deep space, collect stardust, and dodge incoming obstacles.

Technology Behind the Games

Our games are built from the ground up using modern client-side web technologies and game engines, including:

  • Godot Engine: A powerful open-source game engine used to build 2D and 3D games, exported to HTML5 WebAssembly (WASM) for native-like performance in the browser.
  • Matter.js: An advanced 2D physics engine used to simulate realistic rigid body collisions, friction, and bounce for our JavaScript games.
  • HTML5 Canvas: Rendered dynamically using highly optimized drawing operations for smooth frame rates.
