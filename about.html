About Simple Gamez

Simple Gamez is a web gaming portal dedicated to hosting high-quality, instant-play, browser-based games. Our mission is to provide clean, fun, and engaging casual games that load instantly and run smoothly on any device without requiring downloads, installs, or logins.

Our current flagship game is a physics-based merge puzzle game featuring two custom-tailored theme packs:

  • Brain Rot Merge: Drop and evolve iconic memes in a chaotic, bouncy physics box!
  • Football Merge: Progress through your favorite football stars, starting from rising talents like Lamine Yamal, all the way to the ultimate Mickey Mouse Trophy!

Technology Behind the Games

Our games are built from the ground up using modern client-side web technologies, including:

  • Matter.js: An advanced 2D physics engine used to simulate realistic rigid body collisions, friction, and bounce.
  • HTML5 Canvas: Rendered dynamically using highly optimized 2D context drawing operations for smooth frame rates.
  • Responsive Design: Tailored using custom CSS grid and flexbox, ensuring perfect pixel aspect ratios on both wide desktop screens and tall mobile devices.
